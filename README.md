## 🚖 OLA Data Analyst Project

### 📌 Overview

This project focuses on analyzing ride booking data from Ola using SQL to uncover actionable business insights. The analysis covers key metrics such as booking success rate, cancellations, revenue generation, customer preferences, and driver performance.

---

### 🎯 Objectives

* Analyze ride booking patterns and peak demand periods
* Identify reasons for ride cancellations
* Evaluate revenue contribution across vehicle types
* Understand customer behavior and payment preferences
* Measure driver and customer ratings

---

### 🛠️ Tools & Technologies

* SQL (MySQL / PostgreSQL)
* Microsoft Power BI
* Excel

---

### 🧩 Project Structure

* **Data Layer:** SQL views created for reusable and structured data modeling
* **Analysis Layer:** Business queries built on top of views to extract insights
* **Visualization Layer:** Dashboard (Power BI) for interactive reporting

---

### 🔍 Key Analysis Performed

* Successful vs cancelled ride analysis
* Revenue calculation from completed bookings
* Customer segmentation based on ride frequency
* Payment method distribution (UPI, Cash, etc.)
* Vehicle-type performance comparison
* Driver and customer rating analysis

---

### 📈 Key Insights

* Peak ride demand occurs during evening hours
* Certain vehicle types show higher cancellation rates
* UPI is a widely used payment method
* High-frequency customers contribute significantly to revenue

---

### 🚀 Outcome

This project demonstrates end-to-end data analysis workflow — from raw data transformation using SQL views to generating business insights and dashboards. It reflects practical skills required for a Data Analyst role.
